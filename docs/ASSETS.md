# Recursos gráficos del repositorio

| Archivo | Descripción |
|---------|-------------|
| `logo_icon.svg` | Fuente SVG del icono (ondas + nota) |
| `logo_full.svg` | Fuente SVG del logotipo completo |
| `app/src/main/res/drawable/logo_icon.xml` | Vector Drawable del icono (launcher, toolbar) |
| `app/src/main/res/drawable/logo_full.xml` | Vector Drawable del logotipo (pantalla login) |
| `app/src/main/res/drawable/ic_mdi_*.xml` | Iconos MDI — ver [`icons.md`](icons.md) y [`LICENSES.md`](LICENSES.md) |

Los SVG en la raíz son archivos de diseño; los que consume la app están en `res/drawable/`.
